Title: 她
Date: 2025-11-23 19:00:00
Authors: wangzhaohan2910

> 近日，微软发布 Visual Studio 2026，谨以此文献给当初教会我的她。

她是一个我从不曾认识的匆匆过客。

那一年，我在海南海花岛与我的外公团聚，准备过年。吃完年夜饭以后，我们便到楼下放鞭炮。我们带上了先前就在集市上以及商铺里买来的鞭炮，装在塑料袋里，坐电梯下了楼。我们的邻居也下来了。她也来与我们一起放鞭炮。

她与我的母亲年岁相仿，样貌我不记得了，穿着我也不记得了，与我们一样戴着口罩，据说我外公说也是从事软件行业开发工作。

我们在口袋花园里散步，一边放着鞭炮。似曾去了海边，似曾没有去。那是一个最新版 Visual Studio 仍然是 2019 版的年代，我正在琢磨为什么不能像 Dev-C++ 一样，创建一个 .cpp 文件，然后编译。

母亲对我讲，她既然是我的前辈，自然可以跟我探讨探讨。因为我们家里只有我一个是了解这行的，并且我也每天都在做这事。

我问她：“为什么我的 Visual Studio 编译不了文件呀？是不是因为没有登陆微软账号呀？我回去登陆微软账号试试。”后来我想了一想，的确不应该这样问问题，因为这不是疑问句。这也是为什么我现在还能清晰地记得这句话。

她没有急，听我紧张地、口齿不清地把这三句话说完，然后反问我：“你是不是没有创建项目？”

我恍然大悟，答道：“哦！好像是的。我回去试试。”

我与她之间唯一的一次谈话就这样匆匆结束了。

那天，我学会了使用 Visual Studio，也“认识”了一个陌生人。

很久之后，春天来了，我回到了广州。有一天，我不知道正在干什么，好像是母亲给了我四本书，分别是《人月神话》《未来简史》《鸟哥的 Linux 私房菜》和《C++ Primer Plus》。这四本书，我都已经看完了，现存放于我书房里的书架上。

同时对我说：“还记得我们在海南遇见的那位阿姨吗？她送给你的。”

我回答：“哦！是她呀。这样啊。”有没有那个“这样啊”我记不清了。我当时也没想什么。

但是不知道为什么，这么多年过去了，每次在我的眼泪滴下来的前一刻，我总会想到她。
